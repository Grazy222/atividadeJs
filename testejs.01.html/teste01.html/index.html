Crie uma variável chamada “food”. Esta variável deve receber uma string com o nome de uma comida.

Após, crie uma estrutura condicional switch que receba esta variável como condição e que possua três casos e um caso nenhum dos valores for verdadeiro:

caso lasanha for igual a variável, retorne no console: “Não vendemos esta comida aqui”. Caso pizza for igual a variável, retorne no console: “Estamos com escassez de pizza” Caso salada, retorne no console: “Aqui está, são 3 reais”.

Caso nenhuma das opções, retorne no console "não vendemos essa comida aqui" Teste com estas três opções e verifique o console do seu navegador. Crie também um default , que retornará uma mensagem de erro no console.
